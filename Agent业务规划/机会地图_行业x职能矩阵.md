# 机会地图：行业 × 职能（Agent 落地优先级参考）

说明：
- 评分为定性建议：`高 / 中 / 低`（综合“付费意愿、落地难度、集成复杂度、合规风险、ROI 回收周期”）。
- 推荐做法：先按“职能横切”切入，再逐步做行业模板化（如制造/零售/物流）。

## 4) “最建议先做”的 6 个组合（可做产品线）

1. SaaS/互联网：客服工单 Agent（分流 + 草稿 + 质检 + 知识沉淀）
2. 零售/电商：对账差异定位 Agent（订单/物流/退款/渠道结算）
3. 制造：询报价/采购 Agent（询价结构化 + 合同条款核对 + 交期追踪）
4. 物流：异常件处理 Agent（异常分类 + 客户沟通 + 工单写回）
5. 专业服务：交付文档 Agent（会议纪要 → 方案草稿 → 引用来源校验）
6. 横向：销售运营 Agent（会后任务 + CRM 清洗 + 跟进节奏）
